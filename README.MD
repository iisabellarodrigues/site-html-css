# Projeto de Website

## Descrição
Este projeto consiste no desenvolvimento de um website estático utilizando **HTML** e **CSS**, criado como parte de uma atividade acadêmica. O objetivo foi aplicar conceitos fundamentais de desenvolvimento web, incluindo estruturação semântica, estilização, organização de layout e responsividade.

O site apresenta diferentes seções, como introdução, conteúdo informativo, tópicos em destaque, galeria de imagens e informações de contato, proporcionando uma experiência visual agradável e organizada.

## Objetivos do Projeto
- Aplicar conceitos de **HTML5 semântico**.
- Utilizar **CSS3** para estilização e layout.
- Implementar **Flexbox** para organização dos elementos.
- Desenvolver um site **responsivo**, adaptável a diferentes tamanhos de tela.
- Criar uma **galeria de imagens** organizada e visualmente harmoniosa.
- Publicar o projeto utilizando o **GitHub**.

## Tecnologias Utilizadas
- **HTML**
- **CSS**
- **Google Fonts**
- **Flexbox**
